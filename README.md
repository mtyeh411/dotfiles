Based on [dotfiles from paulirish](http://github.com/paulirish/dotfiles).

## Installation
  git clone https://github.com/mtyeh411/dotfiles.git --recurse-submodules && cd dotfiles && ./sync.sh

To update later on, just run `sync.sh` again.

## Manual installs
* `install-deps.sh`
* `.macos`
* `.brew`
